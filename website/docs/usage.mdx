title: Usage
description: How to run Git Proxy in your environment
---
### Run from global install

Once you have followed the [installation](installation) steps, run:

```bash
git-proxy
```

### Using [npx instead of npm](https://www.freecodecamp.org/news/npm-vs-npx-whats-the-difference/)

To directly run Git Proxy without any installation, run:

```bash
npx --package=@finos/git-proxy@1.1.0 -- git-proxy
```
